# Периметр PWA

Полностью автономная HTML/PWA-версия интерактивного триллера «Периметр».

> Возрастное ограничение: 18+. Игра содержит ненормативную лексику, сцены страха, психологического давления, насилия, тяжёлые моральные решения и цензурированные интимные эпизоды между взрослыми персонажами. Откровенные сексуальные детали и нагота не показываются. При первом запуске требуется подтверждение возраста.

## Запуск

PWA требуется HTTP-сервер (service worker не работает через `file://`):

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`. Для публикации достаточно загрузить содержимое этой папки на любой статический HTTPS-хостинг.

## Возможности

- 548 сцен, включая 12 дополнительных напряжённых эпизодов и 6 цензурированных интимных сцен, и 3 акта;
- динамический подбор иллюстраций по месту и типу эпизода — автобус больше не используется как универсальный фон;
- выборы, переменные, документы, разблокировка актов и финалы;
- локальное автосохранение без сервера;
- адаптивный desktop/mobile интерфейс;
- установка на устройство и офлайн-режим;
- карта, архив, журнал и телеметрия;
- поддержка reduced motion и безопасный вывод сюжетных данных.
