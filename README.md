# Периметр PWA

Полностью автономная HTML/PWA-версия интерактивного триллера «Периметр».

> Возрастное ограничение: 18+. Игра содержит ненормативную лексику, сцены страха, психологического давления, насилия и тяжёлые моральные решения. При первом запуске требуется подтверждение возраста.

## Запуск

PWA требуется HTTP-сервер (service worker не работает через `file://`):

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`. Для публикации достаточно загрузить содержимое этой папки на любой статический HTTPS-хостинг.

## Возможности

- 542 сцены, включая 12 дополнительных эпизодов, и 3 акта;
- выборы, переменные, документы, разблокировка актов и финалы;
- локальное автосохранение без сервера;
- адаптивный desktop/mobile интерфейс;
- установка на устройство и офлайн-режим;
- карта, архив, журнал и телеметрия;
- поддержка reduced motion и безопасный вывод сюжетных данных.
