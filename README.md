# Периметр PWA

Полностью автономная HTML/PWA-версия интерактивного триллера «Периметр».

> Возрастное ограничение: 18+. Игра содержит ненормативную лексику, сцены страха, психологического давления, насилия, тяжёлые моральные решения, цензурированную близость и подразумеваемую наготу между взрослыми персонажами. Интимная анатомия полностью скрыта паром, тканью, кадрированием, силуэтом или светом. При первом запуске требуется подтверждение возраста.

## Запуск

PWA требуется HTTP-сервер (service worker не работает через `file://`):

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`. Для публикации достаточно загрузить содержимое этой папки на любой статический HTTPS-хостинг.

## Возможности

- 584 сцены, включая 12 дополнительных напряжённых эпизодов и 42 цензурированные интимные сцены, и 3 акта;
- чтение без постоянной галереи: обычные сцены полностью текстовые;
- 17 ключевых моментов показывают один короткий кинематографичный кадр примерно на 3,2 секунды, после чего он исчезает и не повторяется;
- изображения нельзя выбирать или переключать отдельно от сюжета;
- выборы, переменные, документы, разблокировка актов и финалы;
- локальное автосохранение без сервера;
- адаптивный desktop/mobile интерфейс;
- установка на устройство и офлайн-режим;
- данные и 16 уникальных файлов ключевых кадров доступны офлайн сразу;
- карта, архив, журнал и телеметрия;
- поддержка reduced motion и безопасный вывод сюжетных данных.

## Проверка целостности

Полный отчёт находится в [`reports/QA_50_REPORT.md`](reports/QA_50_REPORT.md). Финальная сборка проходит 39 777 из 39 777 проверок: все сцены и документы достижимы, каждый выбор имеет валидный переход и отдельный результат, время согласовано с фазами сюжета, решения нельзя нажать повторно, а неключевые сцены не запрашивают изображения. Служебные повторы локаций и заголовков, метки «Игрок»/«эхо 2» и черновые названия удалены.
